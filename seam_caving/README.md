SeamCaving
개인적으로 한 문장으로 말하자면 내용 인식 이미지 변환 알고리즘 정도로 말할수 있다.

이미지의 크기를 일반적으로 변환하게 되면 이미지가 왜곡되는 경우가 대부분이다.
이를 해결하기 위해 고안된 방법으로 이미지에서 반복되는 부분, 쉽게 말해 삭제되거나 늘어나더라도 큰 변화를 느끼지 못하는 부분을 변환하는 것이다.
